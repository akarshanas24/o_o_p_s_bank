# Bank Management System (OOP - Python)

A simple console-based Bank Management System implemented using Python Object-Oriented Programming concepts.

## Features

- Create bank account
- Deposit money
- Withdraw money
- Check account balance
- Menu-driven console interface

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Methods
- Composition

## How to Run

1. Clone the repository

git clone https://github.com/akarshanas24/o_o_p_s_bank.git

2. Navigate to the project folder

cd o_o_p_s_bank

3. Run the program

python bank_system.py
